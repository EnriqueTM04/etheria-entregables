Sprint 2

Inicio de Sesión

Esta vista permite a los usuarios iniciar sesión en el sistema ingresando sus credenciales. Es el primer paso para acceder a las funcionalidades del sistema.

Prueba de Aceptación

  1. Ingresar un usuario y contraseña válidos.
  2. Hacer clic en el botón "Iniciar sesión".
  3. Confirmar que se redirige al menú principal si las credenciales son correctas.
  4. Verificar que se muestra un mensaje de error en caso de credenciales incorrectas.

Registro de Usuario

Esta vista permite registrar nuevos usuarios en el sistema proporcionando información personal básica y credenciales.

Prueba de Aceptación

  1. Acceder a la vista de registro.
  2. Completar todos los campos requeridos con información válida.
  3. Hacer clic en el botón "Registrar".
  4. Verificar que el usuario es creado y recibe un mensaje de confirmación.
  5. Intentar registrar un usuario con información incompleta y verificar que se muestra un mensaje de error.

Menú Principal

Esta vista sirve como punto de acceso a las principales funcionalidades del sistema, mostrando opciones como gestión de eventos, usuarios y configuración.

Prueba de Aceptación

  1. Iniciar sesión como usuario válido.
  2. Confirmar que se muestra el menú principal con todas las opciones disponibles.
  3. Seleccionar una opción (por ejemplo, "Gestión de eventos") y verificar que redirige a la vista correspondiente.
  4. Verificar que las opciones no accesibles están deshabilitadas según el rol del usuario.

Lista de Eventos registrados

Esta vista muestra la lista de eventos registrados en el sistema. Permite editar, eliminar y agregar evento.

Prueba de Aceptación

  1. Iniciar sesión como Organizador.
  2. Verificar que se muestra la lista completa de eventos.
  3. Editar, eliminar y agregar un evento.
  4. Confirmar que los cambios se reflejen correctamente.

AgregarEvento

Esta vista muestra la opcion de agregar Evento.

Prueba de Aceptación

  1. Iniciar sesión como Organizador.
  2. Verificar que se muestra los campos correctos.
  3. Llenar el formulario y guardar el evento.
  4. Confirmar que los cambios se reflejen correctamente.

Editar Evento

Esta vista muestra los datos para editar el evento seleccionado.

Prueba de Aceptación

  1. Seleccionar editar evento.
  2. Verificar que se muestran correctamente los datos del evento.
  3. Confirmar cambios y guardar.

Sesiones

Esta vista permite visualizar las sesiones creadas, se pueden editar, eliminar y registrar datos del competidor.

Prueba de Aceptación

  1. Mostrar la lista de los eventos.
  2. Verificar con la base de datos que se muestran correctamente.

Editar Sesion

Esta vista permite a los usuarios editar las sesiones.

Prueba de Aceptación

  1. Seleccionar un evento.
  2. Realizar cambios en los campos deseados.
  3. Guardar los cambios y confirmar que se reflejan en el sistema.

Crear sesion fuera de calendario con Instructor con Privilegios

Esta vista permite a los Instructores con Privilegios agregar sesiones, incluso fuera de calendario.

Prueba de Aceptación

  1. Seleccionar un evento.
  2. Seleccionar la casilla de "Registrar sesión fuera de calendario."
  3. Ingresar los datos correspondientes.
  4. Verificar que se guardó correctamente.

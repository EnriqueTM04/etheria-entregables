Pruebas de Aceptación

Modificación de Fechas de Inscripción y Duración del Evento

Condición

Solo los administradores deben poder modificar las fechas de inscripción y duración de un evento registrado.

Pasos

  1. Iniciar sesión como administrador.
  2. Seleccionar el evento a modificar.
  3. Editar las fechas de inscripción y duración.
  4. Guardar los cambios.

Resultado Esperado

El sistema permite al administrador modificar las fechas de inscripción y duración del evento.

Registro de Datos de Competidores

Condición

El sistema debe permitir el registro de los datos obtenidos por cada competidor en las sesiones de los eventos, incluyendo el número de metros o kilómetros recorridos cuando corresponda.

Pasos

  1. Iniciar sesión como instructor.
  2. Seleccionar el evento y la sesión correspondiente.
  3. Registrar los datos de desempeño de cada competidor.
  4. Guardar los datos.

Resultado Esperado

El sistema permite el registro de los datos de desempeño de cada competidor en las sesiones de los eventos, incluyendo la distancia recorrida cuando corresponda.

Generación de Reportes de Resultados

Condición

El sistema debe generar reportes con los tres mejores lugares de cada evento, por categoría y modalidad, mostrando nombre, distancia/tiempo y posición.

Pasos

  1. Iniciar sesión como administrador.
  2. Seleccionar el evento y la categoría/modalidad a consultar.
  3. Generar el reporte de resultados.
  4. Revisar los tres mejores lugares y sus datos.

Resultado Esperado

El sistema genera un reporte con los tres mejores lugares de cada evento, por categoría y modalidad, mostrando nombre, distancia/tiempo y posición.

Registro de Sesiones Fuera del Calendario

Condición

El sistema debe permitir registrar sesiones fuera del calendario con una razón obligatoria en texto.

Pasos

  1. Iniciar sesión como administrador.
  2. Seleccionar la opción de registrar una sesión fuera del calendario.
  3. Ingresar la razón en texto.
  4. Guardar el registro.

Resultado Esperado

El sistema permite al administrador registrar sesiones fuera del calendario, solicitando una razón obligatoria en texto.

Registro de Eventos Personalizados

Condición

El sistema debe permitir registrar eventos no considerados en la lista predefinida.

Pasos

  1. Iniciar sesión como administrador.
  2. Seleccionar la opción "Agregar evento nuevo".
  3. Ingresar los datos del evento, incluyendo nombre, fechas, modalidades, costos, requisitos y reglas.
  4. Publicar la convocatoria.

Resultado Esperado

El sistema registra y publica correctamente el evento nuevo, permitiendo su inscripción y gestión.

Publicación de Convocatorias

Condición

El sistema debe permitir publicar convocatorias de eventos con la información completa del evento.

Pasos

  1. Iniciar sesión como administrador.
  2. Seleccionar un evento con datos completos.
  3. Revisar la información ingresada (nombre, fechas, costos, requisitos, modalidades, reglas).
  4. Publicar la convocatoria en el portal web.

Resultado Esperado

La convocatoria del evento se publica correctamente y es visible para los usuarios registrados.

Registro de Sesiones Fuera de Calendario para Usuarios con Privilegios

Condición

El sistema debe permitir a usuarios con privilegios de modificación registrar sesiones fuera del calendario, incluyendo una descripción obligatoria del motivo.

Pasos

  1. Iniciar sesión como usuario con privilegios (administrador o instructor autorizado).
  2. Acceder al evento correspondiente desde la lista de eventos.
  3. Seleccionar la opción "Registrar sesión fuera del calendario".
  4. Ingresar la fecha, hora y detalles de la sesión.
  5. Proveer una descripción obligatoria del motivo de la sesión.
  6. Guardar la nueva sesión registrada.

Resultado Esperado

El sistema permite registrar la sesión fuera del calendario, asociándola al evento correspondiente, asegurándose de que solo usuarios con privilegios puedan realizar esta acción.
